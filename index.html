Enable Live Sharing

Enter a GitHub Personal Access Token with gist scope. It's stored only in your browser — never sent anywhere except GitHub's API. Anyone you share the link with will see your live data.

Create one at: github.com → Settings → Developer settings → Personal access tokens → Fine-grained
Required scope: Gists (read and write)
